開發階段（以先達成里程碑為目標）

Phase目標里程碑（做到這裡就有東西可看）
0PDF 可行性驗證產出一份中文不破字、有簽名圖與大章、封面獨立一頁的 PDF
1資料層與後端骨架試算表四張表建好，能用網址建一筆合約單並拿到 token
2簽署頁手機能開連結、看到封面與全文、簽名送出

Phase 0 是停損點。如果 Google Docs 範本這條路產出的中文 PDF 有破字或排版失控， 第二方案是 HTML→PDF 並自行嵌入中文字型；兩條都不行就要回頭找你討論（例如改成產圖片式 PDF）。 沒過 Phase 0 不寫 Phase 3 以後的任何程式碼，避免整套翻掉。

執行方式

  • 一次做一個 Phase，每個 Phase 做完給你看得到的東西（里程碑那欄），不要等全做完才給你。
  • TASK-04 的條文檔是簽署頁與 PDF 範本的同一份文字來源——兩邊各打一次字必然會不一致，這是最容易出的錯。
  • 金額一律以後端 settings 為準重算，前端算的只是給人看的即時數字。
  • 部署類動作全部先本機／測試環境跑通、問過你才推正式。
